# 1. 과제 개요

## (1) 프로젝트 명

**SegmentFlow – 교육영상 세그먼트 기반 학습 퍼널 분석 및 Agent 광고 위치 추천 시스템**

## (2) 프로젝트 목적

교육영상 플랫폼에서 학습 품질 유지와 광고 수익 확보라는 두 목표를 하나의 의사결정 흐름 안에서 연결하는 운영 지원 시스템을 개발한다.

현재 교육영상 운영 환경에서는 다음과 같은 문제가 존재한다.

1. **영상 단위 평균 분석의 한계**: 완주율, 평균 시청시간 등 영상 전체 지표만으로는 어느 구간에서 학습자가 이탈하는지, 어떤 개념 전환 지점이 부담이 큰지 파악하기 어렵다. 세그먼트 단위의 해상도 높은 분석이 필요하다.
2. **광고 운영과 학습 품질 관리의 분리**: 광고 위치가 학습 문맥보다 운영 편의에 의해 결정되어, "수익은 올랐지만 학습 경험이 나빠진" 상태를 뒤늦게 발견하게 된다.
3. **단일 목적 최적화의 한계**: 기존 광고 최적화는 CTR, ROAS 등 수익 지표 중심으로 설계되어 있어, 교육영상에서의 학습 품질 저하를 반영하지 못한다.
4. **설명 가능한 추천의 부재**: 운영자는 "왜 이 위치가 추천되었는지"를 알아야 교육적 맥락에서 의사결정을 내릴 수 있다.
5. **퀴즈 외 학습 상태 파악의 어려움**: 퀴즈만으로는 학습 상태를 충분히 설명하기 어려워, 명시적 신호(이해도 체크 등)와 암묵적 신호(시청 행동 기반)를 함께 보는 구조가 필요하다.

이에 본 프로젝트는 교육영상을 **세그먼트 단위로 분석**하고, **명시적·암묵적 학습 신호를 통합 해석**하여, **학습 흐름을 최대한 보존하는 광고 위치를 Agent가 추천**하는 웹 기반 운영 도구(SegmentFlow)를 개발하는 것을 목표로 한다.

# 3. 기대효과 및 활용방안

## 기대효과

### 기술적 기대효과

1. **세그먼트 단위 분석 기술 확보**: 교육영상을 의미 단위로 자동 분할하고, 세그먼트별 학습 퍼널을 분석하는 기술을 구현함으로써, 기존 영상 단위 평균값 분석의 한계를 극복하는 분석 파이프라인을 확보한다.

2. **다중 학습 신호 통합 분석 체계 구축**: 암묵적 신호(시청 행동 기반)와 명시적 신호(이해도 체크 등 직접 반응)를 통합 해석하는 프레임워크를 구축하여, 퀴즈 유무에 관계없이 학습 상태를 분석할 수 있는 체계를 확보한다.

3. **AI Agent 기반 의사결정 지원 시스템 구현**: LLM/Agent 구조를 활용하여 복수 신호를 종합하고, 설명 가능한 추천 결과와 근거를 제시하는 Human-in-the-loop 추천 시스템을 구현한다.

4. **풀스택 분산 시스템 개발 역량 확보**: Next.js + FastAPI + Celery/RabbitMQ + PostgreSQL 기반의 비동기 분석 파이프라인과 운영 대시보드를 결합한 풀스택 시스템 개발 경험을 축적한다.

### 산업적 기대효과

1. **교육영상 수익화 최적화**: 학습 흐름을 보존하면서도 광고 성과를 확보하는 의사결정 도구를 제공하여, 교육 플랫폼의 수익화 전략 고도화에 기여한다.

2. **운영 효율 향상**: 운영자가 직관이 아닌 데이터 기반으로 광고 위치를 결정할 수 있어, 의사결정 소요시간과 오류를 줄인다.

3. **학습자 경험 보존**: 광고 삽입이 학습 경험에 미치는 부정적 영향을 최소화하여, 교육 플랫폼의 장기적 신뢰성과 사용자 만족도를 유지한다.

## 활용방안

1. **교육영상 플랫폼 운영 도구**: 교육 플랫폼 운영팀이 신규/기존 영상의 광고 위치를 데이터 기반으로 결정하는 표준 운영 도구로 활용
2. **콘텐츠 품질 개선 도구**: 세그먼트별 학습 이탈/이해도 분석 결과를 기반으로 교육 콘텐츠 구조 자체를 개선하는 데 활용
3. **학습 분석(Learning Analytics) 연구 기반**: 세그먼트 단위 학습 퍼널 데이터를 축적하여 교육공학/학습 분석 연구에 활용
4. **취업/창업 포트폴리오**: AI Agent, 풀스택 웹 개발, 데이터 파이프라인 설계 역량을 증명하는 프로젝트로 활용
